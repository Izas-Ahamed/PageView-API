REST API CALL Methods :

  Deployed on Heroku
   
    GET
  - https://pageview-api.herokuapp.com/getviews
    to get view counts and click counts
    
    POST
  - https://pageview-api.herokuapp.com/addview
    to add view count
    
    POST
  - https://pageview-api.herokuapp.com/addclick
    to add click count
  
    DELETE
  - https://pageview-api.herokuapp.com/resetview
    to reset the view count
